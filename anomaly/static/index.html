how to read this: the model only scores after a full 60-second window fills (the warm-up). it outputs a reconstruction error; we map that to an anomaly level 0–100%. when the smoothed level crosses the flag threshold (the red line), the banner reads STRESS FLAGGED. the level is smoothed over ~5 s so single noisy windows don't trip it. ground truth is the real WESAD label, shown so you can check the alignment. the flag is the model's own output; the label is the dataset's truth — the scorecard above counts where they agree (TP/TN) and disagree (FP/FN) live.
